set video size for smaller screen

Description: The video and canvas cannot be view properly on smaller screen,so we change the video size as the screen size reduces

  1. We want a smaller webcam live preview element for smaller screen size, so to know what is the screen size we use windowWidth Learn more about- windowWidth
  2. we already have written if condition ,if windowWidth is less than 800 then your task is to set the video size - width - 300, height- 300 using size() function.Learn more about- size()
  3. else set the video size - width - 500, height- 500 using size()
  4. Write the above code inside setup() in sketch.js
